## Installation Steps

1. **Enable Kiosk Mode**:
   ```bash
   echo "ENABLE_KIOSK_MODE=true" >> .env
   ```

2. **Start the application**:
   ```bash
   docker-compose up -d
   ```

3. **Access kiosk mode**:
   ```
   http://localhost:3000/display
   ```

4. **Install as PWA** (optional):
   - Look for "Install" button in browser
   - Add to home screen for native app experience
